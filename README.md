este proyecto busca tener una mejor facilidad para acceder a los libros, atravez de aplicacion los usuarios podran ver los libros y recervarlos desde casa o escuchar audiolibros disponibles.
para ver el proyecto debes:
- abrir la consola o terminal
- escribir el comando git clone https://github.com/JuanZB360/introducci-n-grupo-4.git para tener este repositorio en tu pc
- abrir el proyecto en visual basic
- ejecutar live server para que se abra el proyecto en el localhost en tu navegador
